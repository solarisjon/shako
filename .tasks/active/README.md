# Active Tasks

This directory contains one file per actively delegated task.

**File naming:** `TASK-XXX.md` (matching the task ID)

**Lifecycle:**
1. Lead creates `TASK-XXX.md` from the task contract template
2. Specialist works on the task
3. Lead reviews and either approves or returns with feedback
4. On approval, the file is deleted (history preserved in git)

**Current state:** Empty — no active tasks.
